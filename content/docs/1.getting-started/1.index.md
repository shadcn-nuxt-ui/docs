## What is shadcn-nuxt-ui?

`shadcn-nuxt-ui` is an open-source project for Nuxt developers using shadcn-vue.

It adds practical templates and reusable components to make implementation easier in real Nuxt projects.

## Why shadcn approach

With the shadcn approach, components are added as source code directly to your project.

That means you can:

- Own the code and customize deeply.
- Avoid vendor lock-in.
- Keep design and behavior aligned with your product.
- Move fast with copyable, composable building blocks.

## Community-driven templates

Templates are built from real implementation needs and shared back to the community.
Some templates include full-stack foundations so you can start from a working base and adapt to your project.

Coverage will continue to expand over time.

## What you get

- More implementation options for Nuxt developers using shadcn-vue.
- Ready-to-use templates for practical app scenarios.
- Reusable components you can install independently.
- A growing set of templates and components, starting with AI Chat-focused blocks.

## How to start

Choose your path based on your goal.

- If you want the fastest path, start with [Templates](/templates).
- If you only need specific building blocks, go to [Components](/docs/components).
